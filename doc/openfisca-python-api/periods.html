class openfisca_core.periods.Period(iterable=(), /)[source]Â¶

Toolbox to handle date intervals.

A period is a triple (unit, start, size), where unit is either âmonthâ or âyearâ, where start format is a (year, month, day) triple, and where size is an integer > 1.

Since a period is a triple it can be used as a dictionary key.
